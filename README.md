# GuessANumberByVelis

## About the project
This is a small console game made with Python.
The program chooses a random number and the player has to guess it.

## What is the goal
The goal of this project is to practice Python basics like:
- user input
- loops
- if statements
- random numbers

## How it works
The program generates a random number.
The player enters guesses.
After every guess the program tells if the number is too high or too low.
The game ends when the correct number is guessed.
